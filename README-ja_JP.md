# night-drive

眠れない夜のために。

GitHub Pages で公開する、静かな夜道ドライブの Canvas 作品です。

MS-DOS 時代の夜道ドライブ作品「FreeWay」と、その Windows 移植版「FreeWay for Win」に着想を得ています。  
参考: 窓の杜（2001-12-18）  
https://forest.watch.impress.co.jp/article/2001/12/18/okiniiri.html
